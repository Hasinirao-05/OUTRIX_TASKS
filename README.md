# OUTRIX_TASKS

This repository contains the tasks completed during my **Outrix Internship**. These projects focus on applying Machine Learning techniques to solve real-world healthcare problems.

## 📂 Repository Structure

```text
OUTRIX_TASKS/
├── Breast Cancer Prediction
└── Diabetes Prediction
```

## 🚀 Tasks

### 1. Breast Cancer Prediction

* Developed a Machine Learning model to classify breast tumors as **Benign** or **Malignant**.
* Performed data preprocessing, feature analysis, model training, and evaluation.
* Used Python and Scikit-learn to build and test the prediction model.

### 2. Diabetes Prediction

* Built a Machine Learning model to predict the likelihood of diabetes based on patient health parameters.
* Applied data preprocessing, exploratory data analysis (EDA), model training, and performance evaluation.
* Implemented using Python and Scikit-learn.
